## üì¶ Local Development
Since this project uses a modern "No-Build" ESM approach, you don't need `npm install` for basic serving.

1. Clone the repository:
   ```bash
   git clone https://github.com/YOUR_USERNAME/nextforge-web.git
   ```
2. Serve the directory using any local server (e.g., Live Server in VS Code).
3. Set your `API_KEY` in your environment to enable AI features.
